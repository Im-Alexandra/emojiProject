SENTIMENT ANALYSIS

Reimagining Sentiment Analysis Through Playful Design or better said: 🧐💬😂❓😀🔍 ➡️😜🎨

This prototype was created as a part of the “Play lab” course in the “Game Design” programme at ITU Copenhagen. It playfully explores and challenges the current dominant discourse of sentiment analysis. If the use of emoji restores emotional framing into computer-mediated human communication, what emotions/sentiment is communicated if we playfully remove the contextual words?

The finished prototype is a simple mobile-friendly web application which includes 2 modes of speech translation: “text translation” and “emoji translation”. Both of these modes measure the sentiment of translated words or emoji.

The prototype also provides a comparison of two different sentiment analysis models. While the “emoji translation” uses a model with manual data labeling, the “text translation” uses a noise labeling model.
